# 扩展webman框架用户插件的功能

## 安装

`composer require ledc/webman-user`

## 扩展的功能

### 用户中间件

- 支持路由参数
- 支持OPTIONS请求的空响应
- 支持JSON响应
- 支持验证数据限制

### 控制器基础类

- 支持数据限制
- 支持自定义数据限制字段
